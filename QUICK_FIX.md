# 快速问题排查指南

## 🔴 遇到"请输入有效数字"错误？

### 立即检查清单

#### 1️⃣ 查看终端日志（最重要！）

启动程序后，终端会显示所有操作日志。找到错误发生时的日志：

```
[UI] 转换失败 - x_pitch_text: '???', 错误: ...
```

这会告诉你具体是哪个输入框、什么内容导致了错误。

---

#### 2️⃣ 手动测试输入

在输入框中：
1. **删除所有内容**（Ctrl+A → Delete）
2. **重新输入数字**（不要复制粘贴）
3. **确保使用半角字符**：
   - ✅ 正确：`0`, `2.5`, `-10`
   - ❌ 错误：`０`, `２．５`, `－１０`（全角字符）

---

#### 3️⃣ 检查默认值

程序启动时的默认值应该可以直接预览：

```
晶圆尺寸：8 inch
参考点：X=0, Y=0
X Pitch：2.00
Y Pitch：2.00
起点位置：左下
行进方向：X方向
```

**直接点击"预览"按钮，应该能成功。**

如果默认值都报错，说明环境有问题。

## 📋 最常见的原因

### 原因1：全角字符（占90%的情况）

**症状：** 输入看起来正常，但无法转换为float

**检查方法：**
```python
# 在Python中测试
s = "２.００"  # 全角
print(len(s))  # 如果大于4，说明有全角字符
```

**解决：** 删除重新输入，或使用：
```python
# 转换全角到半角
s = "２.００".translate(str.maketrans('０１２３４５６７８９．', '0123456789.'))
```

### 原因2：输入法问题

**症状：** 在中文输入法下输入数字

**解决：** 切换到英文输入法

### 原因3：不可见字符

**症状：** 输入框看起来为空但实际不是

**检查方法：** 查看日志中的字符串长度
```
[UI] 原始输入 - ref_x_text: '' (type: str)
```

**解决：** 多次按Delete键，确保完全清空
