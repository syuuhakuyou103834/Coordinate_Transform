# PyQt5版本使用说明

## 新功能特性

本项目已升级使用PyQt5框架，提供更现代化的用户界面和更强大的功能。

### 主要改进

1. **现代化界面**：使用PyQt5构建，界面更美观、响应更快
2. **Excel兼容**：表格数据支持直接复制粘贴到Excel
3. **更好的样式**：表格支持交替行颜色、选中高亮等视觉效果
4. **更好的用户体验**：更直观的布局和操作流程

## 运行程序

### 启动应用
```bash
cd D:\VScode_Project\Coordinate_Transform
.venv\Scripts\python.exe main.py
```

## 复制数据到Excel

这是新版本最重要的功能之一！

### 方法一：复制单行
1. 点击表格中的任意一行进行选中
2. 按下 `Ctrl + C`
3. 打开Excel，选择目标单元格
4. 按下 `Ctrl + V`
5. 数据将以三列形式粘贴（X坐标、Y坐标、膜厚）

### 方法二：复制多行
1. 点击并拖动鼠标选中多行
2. 按下 `Ctrl + C`
3. 在Excel中粘贴即可

### 复制格式说明
- 复制的数据使用**制表符（Tab）**分隔列
- 使用**换行符**分隔行
- 包含表头行（X坐标、Y坐标、膜厚）
- 这种格式与Excel完美兼容

### 示例操作
```
1. 在软件中查询 X坐标 = 0
2. 表格显示3条结果
3. 鼠标点击第一行，按住Shift点击最后一行（选中3行）
4. 按 Ctrl+C
5. 打开Excel，按 Ctrl+V
6. Excel中会出现：
   X坐标    Y坐标    膜厚
   0        63      513.56074
   0        55      516.49051
   ...
```

## 界面布局

### 左侧控制面板
- **文件选择区**：选择CSV文件
- **X坐标查询**：输入X值并查询
- **Y坐标查询**：输入Y值并查询
- **数据摘要**：显示数据统计信息

### 右侧结果面板
- **查询结果表格**：显示查询到的数据
  - 交替行颜色便于阅读
  - 支持行选中
  - 三列：X坐标、Y坐标、膜厚
- **操作提示**：显示复制操作的说明

### 底部状态栏
- 显示当前状态和操作结果

## 快捷键

- `Ctrl + C`：复制选中的表格数据到剪贴板
- `Tab`：在输入框之间切换

## 注意事项

1. **表格聚焦**：只有当表格获得焦点时，Ctrl+C才会复制表格数据
2. **数据精度**：
   - X/Y坐标显示6位小数
   - 膜厚显示5位小数
   - 自动去除末尾多余的0
3. **复制限制**：一次只能复制连续的行

## 与旧版本的区别

| 特性 | Tkinter版本 | PyQt5版本 |
|------|------------|-----------|
| UI框架 | Tkinter | PyQt5 |
| 表格组件 | Treeview | QTableWidget |
| Excel兼容 | 不方便 | 原生支持 |
| 界面样式 | 系统默认 | 自定义样式 |
| 响应速度 | 一般 | 更快 |
| 复制功能 | 需手动处理 | 一键复制 |

## 故障排除

### 问题：Ctrl+C无法复制
**解决方案**：
- 确保点击了表格区域，使其获得焦点
- 确保至少选中了一行数据

### 问题：粘贴到Excel格式不对
**解决方案**：
- 在Excel中使用"选择性粘贴"→"文本"
- 或直接使用Ctrl+V粘贴（Excel会自动识别制表符）

### 问题：程序无法启动
**解决方案**：
```bash
# 重新安装PyQt5
.venv\Scripts\pip.exe install --upgrade PyQt5

# 检查依赖
.venv\Scripts\pip.exe list
```

## 高级用法

### 批量数据处理建议
虽然当前版本一次只能查询一个坐标值，但您可以：
1. 查询第一个坐标
2. 复制结果到Excel
3. 查询下一个坐标
4. 追加粘贴到Excel的下一行
5. 重复以上步骤

### 数据分析建议
1. 将所有需要的数据复制到Excel
2. 使用Excel的数据分析功能：
   - 数据透视表
   - 图表制作
   - 统计分析
   - 条件格式
