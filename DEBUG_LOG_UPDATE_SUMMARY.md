## 使用方法

### 查看实时日志

直接运行程序,日志会自动显示在终端:
```bash
cd D:\VScode_Project\Coordinate_Transform
python main.py
```

### 保存日志到文件

```bash
# Windows CMD
python main.py > debug_log.txt 2>&1

# PowerShell
python main.py | Tee-Object -FilePath debug_log.txt

# Git Bash / Linux
python main.py 2>&1 | tee debug_log.txt
```

### 禁用Debug日志

编辑 `core_fine_map_revision.py`,修改第11行:
```python
DEBUG_MODE = False  # 禁用日志
```
