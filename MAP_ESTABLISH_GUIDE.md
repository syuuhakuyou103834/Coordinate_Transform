# Map建立工具使用指南

## 功能概述

Map建立工具（Map Establish）是一个用于生成晶圆测量Map的实用工具。它可以根据用户指定的参数，自动生成符合圆形区域约束的测量点坐标，并支持蛇形扫描路径。

## 主要功能

### 1. 参数设置

#### 晶圆尺寸
- **4 inch**: 直径100mm，半径50mm
- **6 inch**: 直径150mm，半径75mm
- **8 inch**: 直径200mm，半径100mm

#### 参考点坐标
- 用于唯一确定网格系统
- 所有测量点都是基于参考点加上pitch的整数倍

#### 点间距
- **X Pitch**: X方向相邻两点之间的距离（mm）
- **Y Pitch**: Y方向相邻两点之间的距离（mm）
- 坐标精度会自动与输入的pitch精度保持一致

### 2. 起点位置

Map的扫描起点有四个选项：
- **左上**: (X_min, Y_max)
- **右上**: (X_max, Y_max)
- **左下**: (X_min, Y_min)
- **右下**: (X_max, Y_min)

### 3. 扫描方向

#### X方向（蛇形扫描）
- 从起点开始，保持Y不变，沿X方向移动
- 一行结束后，移动到下一行，X方向反转
- 示例：第1行从左→右，第2行从右→左，第3行从左→右...

#### Y方向（蛇形扫描）
- 从起点开始，保持X不变，沿Y方向移动
- 一列结束后，移动到下一列，Y方向反转
- 示例：第1列从上→下，第2列从下→上，第3列从上→下...

### 4. 圆形约束

所有生成的测量点都必须满足：
```
x² + y² < r²
```
其中 r 是晶圆半径。

## 使用步骤

### 1. 设置参数
```
1. 选择晶圆尺寸（例如：8 inch）
2. 输入参考点坐标（例如：X=0, Y=0）
3. 输入点间距（例如：X Pitch=2.00, Y Pitch=2.00）
4. 选择起点位置（例如：左下）
5. 选择扫描方向（例如：X方向）
```

### 2. 预览Map
```
点击"预览"按钮 → 右侧显示Map预览
```
预览内容包括：
- 晶圆圆形边界
- 所有测量点（蓝色点）
- 起点（红色点）
- 终点（绿色点）
- 扫描路径（蓝色线段）
- 统计信息（总点数、边界范围等）

### 3. 生成CSV文件
```
点击"生成CSV"按钮 → 选择保存路径 → 输入文件名 → 保存
```

生成的CSV文件格式：
```csv
X,Y
-60.00,-60.00
-40.00,-60.00
-20.00,-60.00
...
```

## 技术特点

### 1. 精度控制
- 坐标精度自动与输入的pitch保持一致
- 例如：输入2.00 → 保留2位小数；输入1.5 → 保留1位小数

### 2. 蛇形扫描优化
- 避免重复移动，提高扫描效率
- 支持X方向和Y方向两种扫描模式

### 3. 圆形约束验证
- 确保所有测量点都在晶圆圆形边界内
- 自动过滤边界外的点

### 4. 可视化预览
- 实时显示测量点分布
- 直观展示扫描路径
- 帮助用户验证参数设置

## 注意事项

1. **间距选择**: 间距越小，测量点越多，扫描时间越长
2. **起点位置**: 不同起点会影响扫描路径，但测量点集合相同
3. **扫描方向**: X方向和Y方向生成的点集相同，但顺序不同
4. **文件保存**: CSV文件只包含X和Y两列，不包含膜厚数据

## 与Cross Section功能的配合

1. 使用Map Establish生成测量点CSV文件
2. 进行实际测量，获得膜厚数据
3. 将膜厚数据添加为CSV文件的第三列
4. 使用Cross Section功能查询和分析数据
