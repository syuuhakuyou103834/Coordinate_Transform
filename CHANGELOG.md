# 版本更新日志

## v2.1 - Excel式选择版本 (2025-03-23)

### 🎉 重大更新

本次更新实现了与Excel完全一致的表格选择和操作体验！

### ✨ 新增功能

#### 1. Excel式单元格选择
- **单击选择**：点击任意单元格即可选中
- **Ctrl+点击多选**：按住Ctrl键点击多个不连续的单元格
- **Shift+点击连选**：从起点到终点选择矩形区域
- **Ctrl+A全选**：一键选中表格所有数据

#### 2. 增强的复制功能
- 支持复制任意选中的单元格区域
- 支持复制不连续的单元格
- 支持复制矩形区域
- 自动包含表头信息
- 完美兼容Excel格式

### 🔧 改进优化

#### 表格选择行为
- 从"整行选择"改为"单元格选择"
- 设置SelectionMode为ExtendedSelection
- 实现自定义Shift连续选择逻辑
- 记录最后点击的单元格位置

#### 复制逻辑
- 重写复制函数，支持单元格级复制
- 自动计算选中区域的行列范围
- 保留空单元格信息
- 改进状态提示信息

#### 用户体验
- 更新界面提示文字，说明所有快捷键
- 每次查询后自动重置选择状态
- 优化选择反馈和视觉效果

### 📝 技术细节

#### 代码变更
```python
# 表格选择模式
self.result_table.setSelectionBehavior(QAbstractItemView.SelectItems)
self.result_table.setSelectionMode(QAbstractItemView.ExtendedSelection)

# 添加点击事件处理
self.result_table.itemClicked.connect(self._on_item_clicked)

# 新增快捷键
- Ctrl+A: 全选表格
- Shift+点击: 连续选择
- Ctrl+点击: 多选
```

#### 新增文件
- `EXCEL_SELECTION_GUIDE.md` - Excel式选择功能完整指南

### 🎯 应用场景

1. **精确复制**：只复制需要的单元格，而不是整行
2. **区域复制**：快速复制连续的数据区域
3. **灵活组合**：复制不连续的关键数据点
4. **全量导出**：Ctrl+A快速全选所有数据

### 📊 对比表格

| 功能 | v2.0 | v2.1 |
|------|------|------|
| 选择单位 | 整行 | 单元格 |

| Ctrl+点击 | 不支持 | 多选不连续单元格 |
| Shift+点击 | 不支持 | 连续选择矩形区域 |
| Ctrl+A | 不支持 | 全选所有数据 |

### 📸 功能截图示例

#### 示例1：Ctrl+A全选
```
查询X=0，显示15条结果
→ 点击表格区域
→ 按Ctrl+A
→ 所有单元格被选中
→ 按Ctrl+C复制
→ Excel中Ctrl+V粘贴
```

#### 示例2：Shift连续选择
```
1. 点击第1行第1列
2. 按住Shift
3. 点击第5行第3列
4. 选中5行×3列的矩形区域
```

#### 示例3：Ctrl多选
```
1. 点击第1行第1列
2. 按住Ctrl
3. 点击第3行第2列
4. 点击第5行第3列
5. 选中3个不连续的单元格
```
