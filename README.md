# 地图坐标查询系统

## 项目简介

这是一个基于Python和PyQt5的地图数据处理工具集，包含三个核心功能模块：
1. **Cross Section查询** - 坐标查询和膜厚分析
2. **Map建立工具** - 晶圆Map生成
3. **精细Map修订** - Map可视化编辑和精细修改 ⭐ NEW!

支持读取包含坐标和膜层厚度信息的CSV文件，提供灵活的坐标查询功能，以及完整的Map可视化和编辑能力。通过现代化的图形用户界面，用户可以快速查找特定位置的数据，并支持**一键复制粘贴到Excel**。

## 项目背景

在半导体制造等领域，需要对晶圆或其他基板上的膜层厚度分布进行分析。本工具集旨在：
- 简化坐标数据的查询过程
- 快速生成标准化的Map点阵
- 提供可视化的Map编辑和修改能力
- 帮助用户快速定位特定位置的膜厚信息
- 方便地将结果导出到Excel进行进一步分析

## 功能特性

### 模块1: Cross Section查询 (原地图坐标查询系统)

#### 1. 文件读取
- 支持通过UI界面选择CSV文件
- 自动识别数据列（前两列为X、Y坐标，第三列为膜厚）
- 显示数据加载状态和基本信息

#### 2. 坐标查询
- **X坐标查询**：输入X坐标值，查找所有该X坐标对应的数据点
- **Y坐标查询**：输入Y坐标值，查找所有该Y坐标对应的数据点
- 一次仅可查询一种坐标类型（X或Y）

#### 3. Excel一键复制（核心特色）
- 选中表格数据后按 `Ctrl+C` 即可复制
- 直接在Excel中按 `Ctrl+V` 粘贴
- 自动分成三列，无需手动调整格式
- 包含表头信息
- 支持单行或多行批量复制

#### 4. Excel式选择（v2.1新增）
- **单击**：选中单个单元格
- **Ctrl+点击**：多选不连续的单元格
- **Shift+点击**：连续选择矩形区域
- **Ctrl+A**：全选所有数据
- **操作逻辑**：与Excel完全一致

### 模块2: Map建立工具

#### 1. 晶圆Map生成
- 支持4英寸、6英寸、8英寸晶圆尺寸
- 生成圆形区域内的蛇形扫描路径
- 可配置起点位置（左上/右上/左下/右下）
- 可选X方向或Y方向扫描
- 实时预览扫描路径

#### 2. 参数配置
- 参考点坐标设置
- X/Y方向间距配置
- 自动过滤圆外点位
- 导出标准CSV格式

## 数据格式要求

### Cross Section查询格式
CSV文件应满足以下格式：
- 第一列：X坐标（数值类型）
- 第二列：Y坐标（数值类型）
- 第三列：膜厚值（数值类型）
- 第一行为列名（可选）

示例数据格式：
```csv
X,Y,膜厚
-11,63,512.58415
0,63,513.56074
11,63,505.531
32,55,508.89481
21,55,515.83945
```

### Map建立工具输出格式
生成的CSV文件包含两列：
- 第一列：X坐标
- 第二列：Y坐标

### 精细Map修订格式
**导入格式**: 与Map建立工具相同（X,Y两列）
**导出格式**: 标准CSV，包含索引、X、Y三列

## 注意事项

### 通用注意事项
1. **坐标精度**：系统使用1e-6的容差进行浮点数比较
2. **数据验证**：输入坐标时会验证是否为有效数字
3. **文件格式**：确保CSV文件格式正确
4. **编码格式**：建议使用UTF-8编码保存CSV文件

### 精细Map修订特别注意事项
1. **性能**: 大型Map（>10000点）可能消耗较多内存
2. **撤销**: 当前版本不支持撤销操作，修改前建议保存
3. **备份**: 编辑重要Map前建议先备份原始文件
4. **Debug日志**: 可通过修改`DEBUG_MODE`开关控制日志输出

## 扩展功能建议

未来可以考虑添加以下功能：

### Cross Section模块
1. 支持批量查询多个坐标
2. 导出查询结果到新文件
3. 数据可视化（热力图、散点图等）
4. 添加数据统计分析功能

### Map建立模块
1. 支持自定义扫描路径
2. 更多晶圆尺寸选项
3. 可视化路径编辑

### 精细Map修订模块 ⭐
1. 添加撤销/重做功能
2. 支持批量选择和编辑
3. 添加图形测量工具（距离、角度）
4. 支持导入导出更多格式
5. 添加Map模板功能
6. 优化大型Map的性能
