# 快速入门指南

## 5分钟快速上手

### 第一步：安装依赖（仅需一次）

```bash
# 进入项目目录
cd D:\VScode_Project\Coordinate_Transform

# 激活虚拟环境（PowerShell）
.\.venv\Scripts\Activate.ps1

# 安装依赖（如果还没安装）
pip install -r requirements.txt
```

### 第二步：启动程序

```bash
python main.py
```

### 第三步：加载数据

1. 点击界面上的 **"浏览..."** 按钮
2. 选择 `Map.csv` 文件（或你自己的CSV文件）
3. 等待数据加载完成，左侧会显示数据摘要

### 第四步：查询数据

#### 方法1：查询X坐标
1. 在左侧"X坐标查询"框中输入 `0`
2. 点击 **"查询X坐标"** 按钮
3. 右侧表格显示所有X=0的数据点

#### 方法2：查询Y坐标
1. 在左侧"Y坐标查询"框中输入 `63`
2. 点击 **"查询Y坐标"** 按钮
3. 右侧表格显示所有Y=63的数据点

### 第五步：复制到Excel ⭐（核心功能）

1. 在表格中点击选中一行（或拖动选中多行）
2. 按下 `Ctrl + C`
3. 打开Excel
4. 按下 `Ctrl + V`
5. 完成！数据自动分成三列

## 常见问题

### Q1: 程序无法启动？
```bash
# 检查依赖是否安装
pip list | grep -E "pandas|PyQt5"

# 如果没有，重新安装
pip install -r requirements.txt
```

### Q2: Ctrl+C无法复制？
- 确保点击了表格区域（表格获得焦点）
- 确保至少选中了一行数据

### Q3: Excel粘贴格式不对？
- 直接使用 `Ctrl+V` 粘贴即可
- Excel会自动识别制表符分隔格式

### Q4: 想要测试程序是否正常？
```bash
# 运行测试脚本
python test_app.py
```

## 数据文件格式

你的CSV文件应该长这样：

```csv
X,Y,膜厚列名
-11,63,512.58415
0,63,513.56074
11,63,505.531
...
```

- 第1列：X坐标（数字）
- 第2列：Y坐标（数字）
- 第3列：膜厚值（数字）
- 第1行：列名（可选）

## 提示与技巧

### 提速技巧
1. 使用Tab键在输入框之间快速切换
2. 查询后直接按Ctrl+C，无需手动选中（会复制当前行）
3. 批量查询：查询→复制→查询→复制，最后一次性粘贴到Excel

### 数据分析建议
1. 将多个查询结果复制到Excel的不同工作表
2. 使用Excel的数据透视表分析
3. 制作散点图或热力图可视化
4. 使用条件格式标记异常值

### 键盘快捷键
- `Ctrl+C` - 复制选中的表格数据
- `Tab` - 切换输入框焦点
- `Enter` - 在输入框中触发查询（需点击按钮）
