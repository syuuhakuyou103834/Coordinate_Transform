# Excel式选择功能使用指南

## 功能概述

现在表格支持与Excel完全一致的选择方式！您可以像在Excel中一样选择、复制数据。

## 支持的选择方式

### 1. 单击选择单个单元格
- **操作**：直接点击任意单元格
- **效果**：选中该单元格

### 2. Ctrl+点击 多选不连续单元格
- **操作**：
  1. 点击第一个单元格
  2. 按住 `Ctrl` 键
  3. 继续点击其他单元格
- **效果**：所有点击的单元格都被选中
- **示例**：选择第1行的X坐标、第3行的膜厚、第5行的Y坐标

### 3. Shift+点击 连续选择区域
- **操作**：
  1. 点击起始单元格（例如：第1行第1列）
  2. 按住 `Shift` 键
  3. 点击结束单元格（例如：第5行第3列）
- **效果**：选中从起始到结束之间的矩形区域
- **示例**：选择第1-5行的所有列（3列x5行的矩形区域）

### 4. Ctrl+A 全选表格
- **操作**：
  1. 确保表格获得焦点（点击表格区域）
  2. 按下 `Ctrl + A`
- **效果**：选中表格中所有数据

### 5. Ctrl+C 复制选中内容
- **操作**：
  1. 使用上述任意方式选择数据
  2. 按下 `Ctrl + C`
- **效果**：复制选中区域到剪贴板（Excel兼容格式）

## 实际应用示例

### 示例1：复制整行数据
```
场景：需要复制第3行的所有数据
操作：
1. 点击第3行的任意单元格
2. 按住Shift，点击第3行的第3列（膜厚）
3. 按Ctrl+C
4. 在Excel中粘贴
```

### 示例2：复制多行的特定列
```
场景：需要复制第1、3、5行的X坐标和Y坐标
操作：
1. 点击第1行第1列（X坐标）
2. 按住Ctrl，点击第1行第2列（Y坐标）
3. 继续按住Ctrl，点击第3行第1列和第2列
4. 继续按住Ctrl，点击第5行第1列和第2列
5. 按Ctrl+C
6. 在Excel中粘贴
```

### 示例3：复制整个数据表
```
场景：需要复制所有查询结果
操作：
1. 点击表格区域使其获得焦点
2. 按Ctrl+A（全选）
3. 按Ctrl+C（复制）
4. 在Excel中粘贴
```

### 示例4：复制矩形区域
```
场景：需要复制第2-6行的前两列
操作：
1. 点击第2行第1列（起始单元格）
2. 按住Shift
3. 点击第6行第2列（结束单元格）
4. 按Ctrl+C
5. 在Excel中粘贴
```

## 快捷键总结

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `单击` | 选择单个单元格 | 选中点击的单元格 |
| `Ctrl + 单击` | 多选不连续单元格 | 添加新的单元格到选择区域 |
| `Shift + 单击` | 连续选择区域 | 选中从上次点击到当前位置的矩形区域 |
| `Ctrl + A` | 全选 | 选中表格中所有数据 |
| `Ctrl + C` | 复制 | 复制选中内容到剪贴板（Excel兼容） |

## 复制格式说明

复制到剪贴板的数据格式：
- **列分隔符**：制表符（Tab）- Excel原生支持
- **行分隔符**：换行符
- **包含表头**：自动包含列名行
- **空单元格**：未选中区域复制到Excel会显示为空

## 注意事项

1. **表格焦点**：使用快捷键前，确保表格已获得焦点（点击表格区域即可）
2. **Shift选择**：需要先点击一个单元格作为起点，再按住Shift点击终点
3. **Ctrl多选**：可以添加任意多的不连续单元格
4. **复制精度**：浮点数自动去除末尾多余的0
5. **粘贴位置**：在Excel中粘贴时，选择左上角单元格即可

## 常见问题

### Q1: Shift选择时为什么没有反应？
A: 需要先点击一个单元格作为起点，然后再按住Shift点击另一个单元格。

### Q2: 如何取消已选择的单元格？
A: 点击表格外的任意区域，或按Esc键（如果有）。

### Q3: 复制到Excel后格式不对？
A: 确保使用Ctrl+V直接粘贴，Excel会自动识别制表符格式。

### Q4: 能否选择整列？
A: 可以！使用Shift+点击：点击列的第一个单元格，按住Shift点击该列的最后一个单元格。

### Q5: 复制的表头可以去掉吗？
A: 目前默认包含表头。如不需要，在Excel中删除第一行即可。

## 技术实现

本功能使用PyQt5的QTableWidget实现：
- **SelectionMode**: ExtendedSelection（支持扩展选择）
- **SelectionBehavior**: SelectItems（选择单元格）
- **自定义点击处理**: 实现Shift连续选择逻辑
- **键盘事件**: 处理Ctrl+A和Ctrl+C
