# 简化实现总结 - Map生成算法

## 📋 概述

根据用户的需求，我们实现了一个**更简单、更清晰**的Map生成算法。

### 核心策略

1. **固定起点**：将左上/右上/左下/右下定义为晶圆的4个角点
2. **全区域扫描**：扫描整个正方形区域 `[-r, r] × [-r, r]`
3. **后过滤**：扫描完成后，过滤掉圆外的点
4. **纯净输出**：CSV文件只包含圆内的点

## 🎯 关键改进

### 1. 起点定义（固定4个角点）

```python
def _get_start_point(self, start_position: str, radius: float) -> Tuple[float, float]:
    """获取起点（固定的4个角点）"""
    if start_position == "左上":
        return (-radius, radius)
    elif start_position == "右上":
        return (radius, radius)
    elif start_position == "左下":
        return (-radius, -radius)
    elif start_position == "右下":
        return (radius, -radius)
    else:
        return (-radius, radius)  # 默认左上
```

**优点：**
- ✅ 简单直接，不需要复杂的方向判断
- ✅ 易于理解和维护
- ✅ 起点固定，方便预测

### 2. X方向扫描（扫描整个正方形）

```python
def _generate_x_scan_path_simple(self, start_point, radius, x_pitch, y_pitch):
    """生成X方向蛇形扫描路径（扫描整个正方形区域）"""
    path = []

    # 生成所有Y值（从-r到+r）
    y_values = []
    y = -radius
    while y <= radius + 0.001:
        y_values.append(y)
        y += y_pitch

    # 生成所有X值（从-r到+r）
    x_values = []
    x = -radius
    while x <= radius + 0.001:
        x_values.append(x)
        x += x_pitch

    # 蛇形扫描
    forward = start_x <= 0  # 根据起点X坐标决定第一行方向
    for y in y_values:
        row_x = x_values[:] if forward else reversed(x_values)
        for x in row_x:
            path.append((x, y))
        forward = not forward  # 下一行反向

    return path
```

**关键特性：**
- ✅ 扫描整个正方形区域（不是仅圆内）
- ✅ 蛇形扫描保持（奇偶行反向）
- ✅ 第一行方向根据起点位置智能决定

### 3. Y方向扫描（扫描整个正方形）

```python
def _generate_y_scan_path_simple(self, start_point, radius, x_pitch, y_pitch):
    """生成Y方向蛇形扫描路径（扫描整个正方形区域）"""
    path = []

    # 生成所有X值（从-r到+r）
    x_values = []
    x = -radius
    while x <= radius + 0.001:
        x_values.append(x)
        x += x_pitch

    # 生成所有Y值（从-r到+r）
    y_values = []
    y = -radius
    while y <= radius + 0.001:
        y_values.append(y)
        y += y_pitch

    # 蛇形扫描
    forward = start_y >= 0  # 根据起点Y坐标决定第一列方向
    for x in x_values:
        col_y = y_values[:] if forward else reversed(y_values)
        for y in col_y:
            path.append((x, y))
        forward = not forward  # 下一列反向

    return path
```

### 4. 过滤圆外点

```python
# 在 generate_map_data() 中
scan_path = self._generate_x_scan_path_simple(start_point, radius, x_pitch, y_pitch)
# 过滤掉圆外的点
filtered_path = [(x, y) for x, y in scan_path if x*x + y*y < radius*radius]
# CSV文件只包含圆内的点
return filtered_path, stats
```

**过滤条件：** `x² + y² < r²`

**之前的定义（复杂）：**
- X方向扫描时：
  - 左上 = Y最大行，X最小点
  - 右上 = Y最大行，X最大点
  - 左下 = Y最小行，X最小点
  - 右下 = Y最小行，X最大点
- Y方向扫描时：
  - 左上 = X最小列，Y最大点
  - 右上 = X最大列，Y最大点
  - 左下 = X最小列，Y最小点
  - 右下 = X最大列，Y最小点

**现在的定义（简单）：**
- **无论哪个方向：**
  - 左上 = (-r, r)
  - 右上 = (r, r)
  - 左下 = (-r, -r)
  - 右下 = (r, -r)

**优点：**
- ✅ 定义统一，不依赖于扫描方向
- ✅ 易于理解：就是正方形的4个角

**现在的策略：**
1. 从角点开始

**优点：**

- ✅ 扫描逻辑独立于圆的形状

## 🔍 实现细节

### 蛇形扫描逻辑

**X方向扫描：**
```
起点: (-100, 100) 左上
第1行(Y=98):  (-100, 98) → (-98, 98) → ... → (100, 98)      (左→右)
第2行(Y=96):  (100, 96) → (98, 96) → ... → (-100, 96)       (右→左)
第3行(Y=94):  (-100, 94) → (-98, 94) → ... → (100, 94)      (左→右)
...
```

**Y方向扫描：**
```
起点: (-100, 100) 左上
第1列(X=-98):  (-98, 100) → (-98, 98) → ... → (-98, -100)   (上→下)
第2列(X=-96):  (-96, -100) → (-96, -98) → ... → (-96, 100)  (下→上)
第3列(X=-94):  (-94, 100) → (-94, 98) → ... → (-94, -100)   (上→下)
...
```

### 过滤逻辑

```python
# 圆的方程: x² + y² = r²
# 圆内点满足: x² + y² < r²
filtered_path = [(x, y) for x, y in scan_path if x*x + y*y < radius*radius]
```

**示例：**
- 点 (100, 100): `100² + 100² = 20,000 > 100²` ❌ 圆外，过滤掉
- 点 (18, 98): `18² + 98² = 9,808 < 100²` ✅ 圆内，保留
