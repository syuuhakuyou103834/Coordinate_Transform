## 特色功能详解

### Excel复制粘贴功能

**实现原理**：
1. 监听键盘事件（Ctrl+C）
2. 获取选中行的数据
3. 使用制表符（\t）分隔列
4. 使用换行符（\n）分隔行
5. 设置到系统剪贴板

**优势**：
- Excel原生支持制表符分隔格式
- 粘贴后自动分列
- 保留表头信息
- 无需手动格式化

**使用示例**：
```
软件中：选中3行数据 → Ctrl+C
Excel中：选中A1单元格 → Ctrl+V
结果：
A列(X坐标)  B列(Y坐标)  C列(膜厚)
0          63         513.56074
0          55         516.49051
...
```
